About the Booth

A single-projectionist operation: one student, one course, one deadline, and a lifelong soft spot for the movies.

What this is

The Projection Booth is a film-discovery site built as the Full Stack Development final project. The Home page holds a hand-curated canon of sixteen films with original annotations. The Explore page connects live to The Movie Database (TMDB) API with search, genre filtering, and pagination.

Under the hood

  • Markup Semantic HTML5 across three pages
  • Styling Hand-written CSS3 with Flexbox + Bootstrap 5 grid & navbar
  • Logic ES6 classes only — no jQuery, no legacy syntax
  • Data TMDB REST API + a local curated dataset
  • States Loading skeletons, error retry, and empty states

The custom requirement

My assigned unique UI requirement is a modal popup for a detailed item view. It is implemented from scratch as a DetailModal ES6 class (no Bootstrap modal): a lobby-card style overlay with backdrop dismiss, Escape-key handling, focus trapping, and scroll locking. It powers the detail view on both the Home canon and the live Explore results.
